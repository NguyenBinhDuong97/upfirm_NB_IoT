* Hiện nay tìm hiểu được các ngôn ngữ có thể dùng để lập trình terminal Win App
   bằng các ngôn ngữ sau:
    + Lập trình bằng C#
    + Lập trình bằng C++
    + Lập trình bằng VisualBasic 
    => Cần tìm hiểu và lựa chọn một phương án hợp lý:
 * Sau khi tìm hiểu và lựa chọn các ngôn ngữ để lập trình thì quyết định lựa 
   chọn ngôn ngữ C++ để xây dựng và viết chương trình cho Terminal Winapp bởi:
    + Ngôn ngữ C++ này rất gần với ngôn ngữ C
    + Hiện nay có thể thấy trong lập trình embeded thì các nhà tuyển dụng thường
    yêu cầu biết cả hai ngôn ngữ C và C++
    + Có nhiều thư viện hỗ trợ được viết cho Serial của Window được viết bởi C++
    ví dụ và định hướng sẽ sử dụng WinAPI của window
